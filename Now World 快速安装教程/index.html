Now World 快速安装教程

发表于2022-02-23|更新于2022-02-24|教程

注意，本文档只提供如何使用启动器文档，不提供关于游戏玩法和模组BUG的帮助

整合包下载

在开始的开始请先下载游戏本体 百度网盘

游戏分为 完整包 和 轻量包 两种包体，区别如下：
完整包：包含整合包的所有光影、模组、配置文件以及存档(2022-02-12)
轻量包：仅包含模组和少量配置，没有光影、存档和ffmpeg.exe

除游戏本体外还有 附加包 ，此类包体由完整包拆分而来，不影响游戏包体
附加包仅用于对轻量包的拓展，切勿与完整包一同使用

游戏包体不包含Minecraft本体。因此本整合包不违反”MINECRAFT 最终用户许可协议“

安装整合包

HMCL启动器

启动器下载

首先是PCL启动器的下载地址 点此下载
(需要Java 8u101及以上版本 点此下载)

安装整合包

1.打开启动器，选择 版本列表

2.点击上方的 安装整合包

3.选择 导入本地整合包文件

4.选择下载好的整合包文件并点击打开

5.随便取个名字后点击 安装

PCL启动器

启动器下载

首先是PCL启动器的下载地址 点此下载
(仅支持 Windows 7 至 Windows 11，且需要 .net Framework 4.5 环境)
(需要Java 8及以上版本 点此下载)

安装整合包

1.打开启动器，选择 版本选择

2.选择左边的 导入整合包

3.选择下载好的整合包文件并点击打开

4.随便取个名字后点击 确定
