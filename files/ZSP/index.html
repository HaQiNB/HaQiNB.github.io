我的性格特点

用心向上，充满阳光；遇事能够果断处理，头脑灵活、思维敏捷；敢做敢当、勇于承担重任。

我的座右铭

闪电从不打在相同的地方，人不该被相同的方式伤害两次。

我的优点

为人正直、心地善良、乐于助人。

我的不足

性格方面的弱点，有时给自己压力过大，急于求成，过犹不及。

爱好特长

在校热爱祖国，尊敬师长，团结同学，乐于助人，是老师的好帮手，同学的好朋友。

思想品德

  1. 一直遵守学校的各项规章制度
  2. 热爱生活
  3. 有崇高的理想和伟大的目标
  4. 热爱祖国，尊敬师长，团结同学，乐于助人

学业水平

  1. 努力学习，目的明确
  2. 学习态度端正
  3. 课后按时完成作业
  4. 课上认真听讲

身心健康

  1. 积极参加体育锻炼
  2. 积极参加校开展的各项文体活动
  3. 常参加并组织班级学校组织的各种课内外活动
  4. 养成良好的生活习惯

艺术素养

  1. 审美观念很好，能辨认出什么是真正的美的，也很会表现自己
  2. 对于美的含义有着自己的理解，能够在生活中发现美，懂得美的真正含义
  3. 能够积极参与各类健康的文化艺术活动
  4. 热爱生活，热爱美的事物

劳动与社会实践

  1. 积极参与自愿者活动
  2. 参加公益宣传
  3. 通过社会实践，锻炼实践能力
  4. 了解社会，锻炼自己

自我评语

本人能严格遵守学校纪律，有较强的集体荣誉感，乐于助人，关心同学，与同学相处融洽；学习上刻苦努力，思维活跃。是一个正直诚恳，听话懂事，诚实质朴的学生。老师和同学都很喜欢我。学习上虽然很努力，但学习方法不是最佳，所以成绩还不够理想。希望在以后的学习和生活中，能探索出适合我自己的高效的学习方法。

家长评语
